# 素材经营分析系统 (Material Analyzer)

面向巨量投放团队的本地素材经营分析工具。通过导入 Excel/CSV 数据，自动完成字段识别、数据清洗、素材分级、潜力判断和多维度分析。

## 功能特性

- **数据导入**：支持 Excel (.xlsx) / CSV 文件导入，自动识别巨量导出表头
- **智能清洗**：自动处理空值、百分比、千分位，补算点击率/转化率/转化成本
- **素材分级**：S/A/B/C 四级自动评级，优质/劣质/潜力自动标记
- **批次管理**：同一项目支持多次导入，可区分不同批次，支持批次删除
- **潜力判断**：低消耗素材潜力信号检测，4个维度组合判断
- **审核状态**：支持素材卡审/未卡审筛选和标记
- **项目重命名**：支持项目名称修改
- **清空数据**：一键清空项目内所有素材，保留项目结构
- **多维度分析**：按项目、账户、投放目的分析素材表现
- **剪辑BD分析**：按剪辑和拍摄方案维度分析素材效率
- **推荐上新**：增量/稳量/潜力三个系列推荐
- **可视化看板**：ECharts 图表展示分级分布、消耗占比、账户排行、成本区间
- **数据导出**：一键导出含分级结果的分析数据
- **本地运行**：数据不上传，SQLite 本地存储，安全可控

## 快速开始

### 环境要求

- Python 3.8+
- pip

### 安装依赖

```bash
cd material-analyzer
pip3 install -r requirements.txt
```

### 启动方式

**macOS / Linux：**
```bash
bash start.sh
```

**Windows：**
```bash
# Git Bash / PowerShell
bash start_windows.sh

# 或 CMD
pip install -r requirements.txt
python app.py
```

**直接运行：**
```bash
python3 app.py
```

启动后浏览器访问 **http://localhost:8080**

> **注意**：macOS 可能占用 5000 端口，本项目默认使用 **8080** 端口。

## 使用流程

1. **创建项目** → 顶部「+ 新建项目」按钮
2. **导入数据** → 左侧「数据导入」页，拖拽或选择巨量导出的 Excel/CSV 文件
3. **查看看板** → 左侧「项目看板」查看总览、分级分布、账户排行、成本区间
4. **素材分析** → 左侧「素材分析」按等级/账户/投放目的/审核状态筛选素材
5. **推荐上新** → 左侧「推荐上新」查看增量/稳量/潜力推荐系列
6. **剪辑分析** → 左侧「剪辑BD分析」按剪辑和方案维度分析
7. **调整规则** → 左侧「分级规则」修改分级参数，重新计算
8. **清空数据** → 项目顶部「清空数据」按钮，保留项目删素材

## 素材分级规则

### 四级评级

| 等级 | 核心指标 | 含义 |
|------|----------|------|
| **S级** | 消耗累计TOP 10% 且 转化成本达标 | 头部高效素材，加大投放 |
| **A级** | 消耗累计TOP 30% 且 转化成本达标 | 优质素材，持续关注 |
| **B级** | 有消耗有转化但未达S/A标准 | 中等素材，优化方向 |
| **C级** | 无转化或消耗极低 | 低效素材，考虑暂停 |

### 特殊标记

| 标记 | 判断逻辑 | 含义 |
|------|----------|------|
| **优** | 消耗>中位数且转化成本<中位数 | 优质素材，ROI好 |
| **劣** | 消耗<中位数且转化成本>中位数 | 劣质素材，效率低 |
| **P** | 低消耗但CTR异常高，潜力待验证 | 潜力素材，值得测试 |

### 潜力参数

- `potential_cost_max` = 500（消耗上限）
- `potential_ctr_mult` = 1.5（CTR倍数）
- `potential_min_show` = 1000（最低展示量）
- `potential_min_click` = 20（最低点击量）

所有参数均可在「分级规则」页面调整。

## 支持的字段

系统自动识别以下巨量导出字段（模糊匹配）：

- 素材：素材名称、素材ID、素材类型
- 账户：账户ID、账户名称
- 投放：计划名称、投放目的、广告组名称
- 指标：消耗、展示、点击、点击率、转化、转化成本、转化率、ROI、深度转化
- 审核：审核状态（审核通过/审核不通过）
- 其他：日期、落地页链接、图片/视频链接

缺失的指标字段会自动补算（如点击率 = 点击/展示）。

### 视频名称格式

系统自动解析：`编号+价格+产品+剪辑+BD+文案关键词.mp4`

### 文件名格式

`视频库_账户ID_导出时间.csv`

## 常见问题

**Q: 导入文件后字段没有正确识别？**
A: 检查表头是否包含关键词（如"消耗"、"展示"等），系统基于关键词模糊匹配。导入前可在预览页查看识别结果。

**Q: 如何修改分级规则？**
A: 左侧「分级规则」页面可编辑参数，修改后点击「重新分级」即可应用到所有素材。

**Q: 数据存在哪里？安全吗？**
A: 所有数据存储在本地 `data/analyzer.db`（SQLite文件），不会上传到任何服务器。

**Q: 端口8080被占用怎么办？**
A: 在 `app.py` 中搜索 `8080` 并修改为你需要的端口号。

**Q: 清空数据和删除项目有什么区别？**
A: 「删除项目」会删除整个项目结构；「清空数据」保留项目，但删除该项目下所有素材、账户和批次记录，方便重新导入。
